=summary
<p>Like
<tt><a href="#duk_compile">duk_compile()</a></tt>, but the compile input
is given as a C string.  The filename associated with the function is
automatically provided from the <tt>__FILE__</tt> preprocessor define
of the caller.</p>

=example
duk_compile_string(ctx, 0, "print('program code');");
