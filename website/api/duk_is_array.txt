int duk_is_array(duk_context *ctx, int index);

=stack
[ ... val! ... ]

=summary
<p>Returns 1 if value at <tt>index</tt> is an object and is an Ecmascript array
(has the internal class <tt>Array</tt>), otherwise returns 0.  If <tt>index</tt>
is invalid, also returns 0.</p>

<p>This function returns 1 when the following Ecmascript expression is true:</p>
<pre class="ecmascript-code">
Object.prototype.toString.call(val) === '[object Array]'
</pre>

=example
if (duk_is_array(ctx, -3)) {
    /* ... */
}
