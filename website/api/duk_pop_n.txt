=proto
void duk_pop_n(duk_context *ctx, duk_uint_t count);

=stack
[ ... val1! ...! valN! ] -> [ ... ]

=summary
<p>Pop <code>count</code> elements off the stack.  If the stack has fewer than
<code>count</code> elements, throws an error.  If <code>count</code> is zero, the
call is a no-op.</p>

=example
duk_pop_n(ctx, 10);

=tags
stack
