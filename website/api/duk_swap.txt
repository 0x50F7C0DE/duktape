=proto
void duk_swap(duk_context *ctx, int index1, int index2);

=stack
[ ... val1! ... val2! ... ] -> [ ... val2! ... val1! ... ]

=summary
<p>Swap values at indices <tt>index1</tt> and <tt>index2</tt>.  If the
indices are the same, the call is a no-op.  If either index is invalid,
throws an error.</p>

=tags
stack

=example
duk_swap(ctx, -5, -1);
