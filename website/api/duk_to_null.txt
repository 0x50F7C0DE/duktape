=proto
void duk_to_null(duk_context *ctx, int index);

=stack
[ ... val! ... ] -> [ ... null! ... ]

=summary
<p>Replace the value at <tt>index</tt> with <tt>null</tt>, regardless
of the previous value.  Throws an error if index is invalid.</p>

<div class="note">
Equivalent to <tt>duk_push_null()</tt> followed by <tt>duk_replace()</tt>
to target index.
</div>

=example
duk_to_null(ctx, -3);

=tags
stack
